## Features

- Hinglish AI conversations
- Funny low-energy humor
- Calm savage replies
- Server-aware contextual responses
- Reads recent chat context
- Remembers small conversation history
- Discord member style personality
- Reply / mention aware
- Sticker & emoji confusion reactions
- Lightweight Gemini 3.1 Flash Lite setup
- Fast response optimized
- Random chaos mode occasionally
